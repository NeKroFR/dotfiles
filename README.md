# Installation

First ensure [stow](https://www.gnu.org/software/stow/) is installed.

```
git clone https://github.com/NeKroFR/dotfiles.git ~/dotfiles
cd ~/dotfiles
bash install-requierments.sh
bashh apply.sh
```
